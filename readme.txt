== Description and Use ==

Simple redirection from pages or posts using Custom Fields.

  1. On the page or post you wish to redirect from, open up the Custom Fields section
  2. Type in 'redirect' for the key, and then any URL for the value
  3. Press Add Field, save the post, and you're done!

== Installation ==

1. Upload whole redirection folder to the /wp-content/plugins/ directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. How it Works?
3.1. On the page or post you wish to redirect from, open up the Custom Fields section
3.2. Type in 'redirect' for the key, and then any URL for the value
3.3. Press Add Field, save the post, and you're done!
